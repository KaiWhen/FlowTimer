Flowtimer with modifications to IGT Tracking to be used with [Red Extended Path Tool](https://github.com/KaiWhen/RedExtTool).

#### Changes
- Visual timer shows current IGT second based on offset and delayers
- Add second failure textbox to IGT Timers
- Skip beeps on second failures
- Read current target frame and IGT seconds from Red Extended Path Tool while flowtimer is running
- Allow current target frame to be adjusted with hotkeys
